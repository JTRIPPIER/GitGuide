# Git Reference

#### Pulling
```
  git pull
```


#### Pushing
```
  git push
  git push -f (Force)
```

#### Rebasing
```
  git rebase 
```

#### Merging
```
  git merge 
```

#### Tidy up 
Force clean remove untracked files and directories
```
  git clean -df
```

#### Others 
Search the logs
```
  git log -S <search-string>
```
